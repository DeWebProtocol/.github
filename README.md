# DeWebProtocol .github

This repository maintains organization-level GitHub configuration and community
documentation for DeWebProtocol.

It contains:

- the organization profile shown on the DeWebProtocol GitHub organization page;
- common contribution guidance;
- the Code of Conduct;
- security reporting guidance;
- support and repository-routing guidance.

Project-specific documentation, code, tests, benchmarks, and implementation
notes live in the relevant project repositories.
